# Project Roles & Personas Coverage Checklist

- [ ] Product Manager responsibilities assigned and documented
- [ ] Project Manager role confirmed for scheduling/risk/output
- [ ] Developer(s) named and onboarded
- [ ] QA Lead/test responsibilities defined
- [ ] UX Designer included for any user-facing change
- [ ] Scrum Master or equivalent facilitator confirmed
- [ ] Business Analyst engaged for requirement-heavy projects
- [ ] Stakeholder list with contact/engagement plan
- [ ] All new roles aware of key artifacts and process docs
